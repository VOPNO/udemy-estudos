Oque é um Pseudo Elemento?

É um recurso que permite estilizar partes específicas de um elemento HTML.

Sintax de um Pseudo Elemento

Seletor::pseudo-elemento {
        propridade: valor;
            }

Principais tags de Pseudos Classes

::after

::before

::first-letter

::first-line

::selection

Oque cada tag faz

::after

O pseudo-elemento ::after é usado para inserir conteúdo após o conteúdo do elemento selecionado.
Exemplo:

::before

O pseudo-elemento ::before é semelhante ao ::after, mas é usado para inserir conteúdo antes do conteúdo do elemento selecionado.

::first-letter

O pseudo-elemento ::first-letter seleciona a primeira letra dentro de um elemento.

::first-line

O pseudo-elemento ::first-line seleciona a primeira linha de texto dentro de um elemento.

::selection
